# Salário x Inflação — Dashboard Web

Dashboard econômico responsivo criado com **HTML, CSS, JavaScript e Chart.js** para analisar a evolução do salário mínimo e da inflação IPCA entre 2010 e 2020.

## Funcionalidades

- Cards com os principais indicadores do período
- Evolução do salário mínimo em gráfico de linha
- Comparação entre reajuste salarial e IPCA
- Gráfico de saldo real por ano
- Filtro para consultar um ano específico
- Tabela anual completa
- Tema claro e escuro com preferência salva
- Impressão do relatório
- Layout responsivo para celular, tablet e desktop
- Estrutura acessível com HTML semântico, foco visível e suporte a teclado

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Chart.js 4.4.9

## Como executar

Como a aplicação não exige build, há duas opções:

### Abrir diretamente

Abra o arquivo `index.html` no navegador.

### Usar o Live Server

No Visual Studio Code:

1. Instale a extensão **Live Server**.
2. Clique com o botão direito em `index.html`.
3. Selecione **Open with Live Server**.
